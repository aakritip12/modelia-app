This project is based on node v22.11.0

# To Prettify the frontend project in one go us
```
  cd ./packages/frontend
  yarn prettier --write .

```

# To Run the Frontend
```
  cd ./packages/frontend
  pnpm install
  pnpm dev:frontend
```

# To Run the Backend
```
  cd ./packages/backend
  pnpm install
  pnpm dev:backend
```
